## Описание проекта

Проект представляет собой интеллектуальный сервис агрегации новостей с элементами персонифицированной выдачи информации.

Реализовано:
- Классификация новостей (LinearSVC, Accuracy ≈ 0.81)
- Персонализированные рекомендации
- Генерация краткого содержания (ruT5)
- Анализ распределения данных
- Логирование действий пользователя
- История обучения модели

## Структура

- notebooks/ — обучение модели
- models/ — сохранённые модели
- data/ — датасет
- main.py — GUI Streamlit
